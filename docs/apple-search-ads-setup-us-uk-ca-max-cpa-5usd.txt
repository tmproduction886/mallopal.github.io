Apple Search Ads (ASA) Setup Playbook — US/UK/CA — Max CPA $5 — No CPP (yet)
Mallo / MalloPal
Last updated: 2026-03-06

Goal
- Primary: Maximize NEW iOS installs while keeping Cost per Acquisition (CPA) <= $5.
- Secondary: Learn winning search terms fast, then scale only what hits CPA.

Assumptions / constraints
- Markets: United States, United Kingdom, Canada.
- No Custom Product Pages (CPP) yet (we’ll use the default product page).
- Product positioning: warm AI companion; proactive check-ins; persistent memory; weekly recap.
- Avoid positioning as therapy, dating, romance, roleplay, or clinical support.

What “success” looks like (minimum viable)
- Brand campaign: CPA <= $2, TTR >= 8%.
- Generic Core: CPA <= $5, with at least 10–30 winning exact-match terms per market by day 14.
- Discovery: produces 5–15 new converting terms per market by day 14.

============================================================
1) Account structure (best-practice, simple, scalable)
============================================================
Create the SAME campaign structure per country.

Country campaign naming convention
- [US] | Brand | Exact
- [US] | Generic Core | Exact+Broad
- [US] | Competitor | Exact
- [US] | Discovery | Search Match + Broad
Repeat for UK, CA:
- [UK] | ...
- [CA] | ...

Why this structure works
- Brand defends your name cheaply.
- Generic Core is your main growth engine (high intent).
- Competitor is optional, tightly controlled to avoid wasting CPA.
- Discovery is a “keyword mining machine” (small budget; converts -> move to Core exact).

============================================================
If you have a fixed daily budget, use the percentages below per country.
If you don’t, start with these absolute numbers (per country) and adjust after day 3.

Recommended starting daily budgets (per country)

Budget allocation within each country

- Competitor: 10% (US $3 / UK $1.5 / CA $1.5)

- Pause Competitor until Core has stable winners under $5.

============================================================
3) Bid plan (starting CPT bids, tuned for Max CPA $5)
============================================================
Key idea
- Without CPP, you need tighter control: exact-match terms should carry the spend.
- Start conservative; raise bids ONLY on terms meeting CPA and showing conversion momentum.

Set your “stop-loss” threshold
- If a keyword spends 1.5x your target CPA ($7.50) with 0 installs, it is a candidate to pause or negate.
- If a search term spends $10 with 0 installs, negate it (unless it’s extremely relevant and early).

Starting bids (per match type)
- Brand Exact: higher bid to win impression share; usually still cheap.
  - US: $1.20
  - UK: £0.85
  - CA: C$1.25

- Generic Core Exact: moderate
  - US: $0.90
  - UK: £0.65
  - CA: C$0.95

- Generic Core Broad: -25% vs Exact
  - US: $0.70
  - UK: £0.50
  - CA: C$0.75

- Discovery Search Match: lowest
  - US: $0.45
  - UK: £0.35
  - CA: C$0.50

- Competitor Exact: low-to-moderate; watch CPA hard
  - US: $0.60
  - UK: £0.45
  - CA: C$0.65

Bid adjustment rules (simple)
- If CPA <= $4 AND >= 3 installs in last 7 days -> increase bid +10% to +20%.
- If CPA between $4–$5 with >= 3 installs -> keep bid; optimize negatives + creatives later.
- If CPA > $5 with >= 2 installs -> reduce bid -15% to -25%.
- If 0 installs and spend >= $7.50 -> pause keyword OR add negative exact for that search term.

============================================================
8) Operational cadence (how to run this daily)
============================================================
Day 0 (setup day)
- Launch US campaigns first, then clone to UK and CA (same structure).
- Add negative list to Generic + Discovery from day 0.
- Set budgets and starting bids as above.

Days 1–3 (stop leaks, get signal)
Daily (10 minutes per country):
1) Open Search Terms report (Discovery + Broad ad groups first).
2) For any irrelevant term: add Negative Exact immediately.
3) For any term with spend >= $7.50 and 0 installs: pause keyword or negate that term.
4) Identify any term that got installs with CPA <= $5.
   - Add that term as Exact in Generic Core (right intent ad group).
   - Add Negative Exact for that term in Discovery/Search Match to force routing to Exact.

Days 4–7 (build the winner set)
Every 48 hours:
- Move all converting search terms into Generic Core Exact.
- Lower bids on Broad/Search Match if they exceed CPA.
- Increase bids +10–20% on Exact terms with CPA <= $4 (and >= 3 installs/7d).

Days 8–14 (scale safely)
Weekly:
- Prune losers: pause exact keywords with CPA > $5 consistently.
- Expand winners: add close variants (plural/singular; “check-in” vs “check in”).
- Consider adding the optional “loneliness/living alone” set with low bids; pause fast if it hurts CPA.

============================================================
9) When to add Custom Product Pages (CPP) (your biggest next lever)
============================================================
You said “not custom page yet,” but here’s the trigger:
- Once you find 15–30 exact terms converting in Generic Core, create 3 CPP and map:
  - AI companion terms -> CPP emphasizing “checks in + remembers”
  - Check-in/journal terms -> CPP emphasizing “daily question ritual”
  - Weekly recap/reflection terms -> CPP emphasizing “weekly recap”

Expected impact
- CPP typically increases CVR, which lowers CPA, which lets you bid higher and get more volume under the same CPA target.

============================================================
10) Quick troubleshooting
============================================================
Problem: CPA too high across the board
- Reduce bids 15–25% on Generic Broad + Search Match immediately.
- Tighten negatives (especially “dating/roleplay/therapy” terms).
- Shift spend to Exact terms only until CPA stabilizes.

Problem: Lots of taps, low conversions (low CVR)
- Your default App Store page may not match the intent of the keywords.
- Narrow keyword set to “companion/check-in/weekly recap” that matches your first screenshots.

Problem: Not enough volume
- Raise bids on the best exact terms (+10–20%).
- Add more broad variants, but keep bids low and negatives strong.

Problem: UK/CA not spending
- Increase budgets slightly and raise Exact bids by +10–15% only on terms already converting in US.
